# Page Builder

The page-builder module (13) assembles landing pages from reusable sections — no theme code required.

## What you get

- **Section library:** hero, countdown, features, testimonials, FAQ, comparison, and CTA sections.
- **Page templates:** ready-made layouts such as a **Coming Soon** page (hero + countdown + CTA).
- **Theme blocks** for each section so pages live inside your theme.

## Surfaces

- **Theme blocks:** `page-hero`, `page-countdown`, `page-features`, `page-testimonials`, `page-faq`, `page-comparison`, `page-cta`.
- **Admin:** the Page Builder, including template selection (e.g. `coming_soon`).

## FAQ

**Are these real pages or popups?**
Real, server-rendered sections placed in your theme through app blocks — crawlable and themable.

**Can I build a coming-soon / pre-launch page?**
Yes — the Coming Soon template combines a hero, countdown, and CTA out of the box.
